# Doğum Günü Papatyası

İki aşamalı, mobil uyumlu doğum günü kutlama sayfası. Dört papatya yaprağının her biri farklı bir mesaj gösterir ve yalnızca bir kez seçilebilir.

## Yerelde çalıştırma

Projeyi VS Code Live Server ile açabilir veya bu klasörde şunu çalıştırabilirsiniz:

```bash
npx serve .
```

## Vercel'e yayınlama

Klasörü GitHub'a gönderip Vercel'de **New Project** ile içe aktarın. Framework seçimi gerekmez; proje statik olarak otomatik yayınlanır.
